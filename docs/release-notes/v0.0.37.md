## 呆呆面板 Flutter App v0.0.37

### 更新内容
- 对接后端 v2.2.17 版本新功能
- 任务创建和编辑新增 Python 版本选择（3.10 / 3.11 / 3.12），支持任务级 Python 运行环境
- 依赖管理页面新增 Python 版本筛选，可按 3.10 / 3.11 / 3.12 版本查看和管理依赖
- 任务卡片显示 Python 版本标签
- 修复数据备份与恢复功能：支持以 JSON 文件形式导出和导入
  - 导出：选择保存位置生成带时间戳的 JSON 备份文件
  - 导入：选择备份文件直接导入，无需手动粘贴
- 持续优化深色模式显示

### 下载说明
- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装
