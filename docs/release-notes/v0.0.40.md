## 呆呆面板 Flutter App v0.0.40

### 更新内容
页面重构：
- **设置页面**：合并用户管理和个人设置，统一管理入口
- **仪表盘**：集成系统监控信息，包括 CPU/内存/磁盘使用率、趋势图表
- 修复个人设置用户信息显示"未知用户"问题

界面优化：
- 移除 AppBar 中多余的刷新按钮
- 简化导航结构，从 13 个 Tab 精简为 10 个

### 导航结构
```
仪表盘 → 任务 → 环境变量 → 依赖管理 → 脚本 → 日志 → 订阅 → 开放API → 安全 → 设置
```

### 下载说明
- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装
