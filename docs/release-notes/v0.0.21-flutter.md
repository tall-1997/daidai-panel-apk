## 呆呆面板 Flutter App v0.0.21-flutter

### 更新内容
- 任务日志改为每1秒实时刷新，动态获取即时显示
- 添加实时刷新指示器（绿色圆点 + 实时标签）
- 优化日志更新逻辑，只在内容变化时更新UI
- 修复 Android 构建问题：更新 compileSdk 到 36
- 修复 core library desugaring 支持
- 更新 GitHub Actions 构建配置

### 下载说明
- **Android**: 下载 `app-release.apk` 安装使用
- **iOS**: 下载 `daidai-app.ipa`，通过 AltStore 或 Sideloadly 安装

### 技术栈
- Flutter + Dart + Provider
- 支持 Android 和 iOS 平台
