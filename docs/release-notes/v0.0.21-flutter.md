## 呆呆面板 Flutter App v0.0.21-flutter

### 更新内容
- 修复 Android 构建问题：使用 CommonExtension 替代已弃用的 BaseExtension
- 修复 compileSdk 设置：强制所有插件使用 compileSdk 36
- 修复 Gradle projectsEvaluated 回调问题
- 更新 App 图标为新图片
- 创建 release notes 文件支持自动构建

### 下载说明
- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装

### 技术改进
- 使用 `com.android.build.api.dsl.CommonExtension` 替代 `com.android.build.gradle.BaseExtension`
- 使用 `plugins.withId` 方式配置 Android 插件的 compileSdk
- 确保所有 Flutter 插件使用统一的 compileSdk 36
