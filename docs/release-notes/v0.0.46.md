## 呆呆面板 Flutter App v0.0.46

### 更新内容
新增功能：
- **日志导出**：添加 App 调试日志导出功能，方便问题排查
  - 支持 JSON 和文本两种导出格式
  - 可通过系统分享功能发送日志文件
  - 支持清除本地缓存日志

功能改进：
- 仪表盘使用 LogService 记录 API 调用日志
- 设置页面添加日志管理入口

### 使用方法
进入 设置 > 导出调试日志，选择导出格式即可分享日志文件。

### 下载说明
- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装
