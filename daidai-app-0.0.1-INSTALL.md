# 呆呆面板 Android App v0.0.1 安装说明

## 下载

下载 `daidai-app-0.0.1-debug.apk` 文件到您的 Android 设备。

## 安装

1. 在 Android 设备上打开文件管理器
2. 找到下载的 APK 文件
3. 点击 APK 文件开始安装
4. 如果出现"未知来源"警告，请前往设置允许安装未知应用
5. 完成安装

## 首次使用

1. 打开应用
2. 输入面板地址（例如：http://192.168.1.100:5700）
3. 输入用户名和密码
4. 点击登录

## 功能说明

### 基础功能
- **任务管理**：查看、执行、停止定时任务
- **环境变量**：查看和管理环境变量
- **脚本管理**：查看和编辑脚本文件
- **执行日志**：查看任务执行日志
- **系统设置**：查看系统信息

### 扩展模块
- **Web助手**：网页Cookie提取和导入（开发中）

## 系统要求

- Android 8.0 (API 26) 或更高版本
- 网络连接到呆呆面板服务器

## 注意事项

1. 这是 debug 版本，仅用于开发和测试
2. 部分功能尚未完整实现
3. 需要配置正确的面板地址才能正常使用
4. 建议在 Wi-Fi 环境下使用
