# 标签打印工具

一个简单易用的 Windows 桌面标签打印应用。

## 功能特点

- ✅ 自定义标签模板
- ✅ 快速输入数据并打印
- ✅ 批量打印支持
- ✅ 数据导入（JSON/CSV）
- ✅ 打印历史记录
- ✅ 标签预览

## 快速开始

### 方式一：直接运行 Python 脚本

1. 确保已安装 Python 3.8+
2. 安装依赖（可选，用于增强打印功能）：
   ```bash
   pip install pywin32
   ```
3. 运行应用：
   ```bash
   python label_printer.py
   ```

### 方式二：打包成 Windows exe

1. 双击运行 `build_windows.bat`
2. 等待打包完成
3. 在 `dist` 目录中找到 `标签打印工具.exe`

## 使用说明

### 快速打印

1. 选择模板
2. 输入数据
3. 点击"打印标签"

### 批量打印

1. 切换到"批量打印"选项卡
2. 选择模板
3. 输入数据（JSON 或 CSV 格式）
4. 点击"批量打印"

### 模板管理

1. 点击右上角"模板管理"按钮
2. 新建/编辑/删除模板
3. 定义字段和模板内容

### 数据格式

**JSON 格式：**
```json
[
  {"姓名": "张三", "电话": "1234567890"},
  {"姓名": "李四", "电话": "0987654321"}
]
```

**CSV 格式：**
```
姓名,电话
张三,1234567890
李四,0987654321
```

## 模板占位符

在模板内容中使用 `{字段名}` 作为占位符：

```
商品: {商品名称}
价格: ¥{价格}
条码: {条形码}
```

## 配置文件

配置文件 `label_config.json` 保存在应用目录下，包含：
- 模板定义
- 字段配置
- 模板内容

## 常见问题

### Q: 打印机无法打印？

A: 
1. 确保打印机已连接并设置为默认打印机
2. 检查打印机驱动是否安装
3. 尝试先"预览标签"，然后手动打印

### Q: 如何添加新模板？

A:
1. 点击"模板管理"
2. 点击"新建模板"
3. 填写模板名称、尺寸
4. 添加字段
5. 编辑模板内容（使用 `{字段名}` 占位符）
6. 保存

### Q: 批量打印数据格式错误？

A:
- JSON 格式必须是数组，每个元素是一个对象
- CSV 格式第一行必须是表头
- 字段名必须与模板中的字段名一致
