# 自动化脚本集合

本目录包含常用的自动化脚本，支持青龙面板运行。

## 抢元宝派系列

| 脚本 | 说明 | 定时规则 |
|------|------|----------|
| `抢元宝派-优化版.py` | 基础异步协程版，50并发 | `58 11,19 * * *` |
| `抢元宝派-代理池版.py` | 内置23个代理，自动检测 | `58 11,19 * * *` |
| `抢元宝派-实时代理版.py` | 从6个代理源实时抓取 | `58 11,19 * * *` |
| `抢元宝派-全时段验证版.py` | 全时段代理验证，缓存机制 | `*/5 * * * *` |

### 环境变量
- `YUANBAO_COOKIE`: 元宝派 Cookie，多个账号用 `&` 分隔
- `CUSTOM_PROXIES`: 自定义代理，格式 `ip:port`，多个用逗号分隔

### 使用方法
1. 在青龙面板中添加脚本
2. 设置环境变量 `YUANBAO_COOKIE`
3. 按建议设置定时规则

## 酷我音乐

| 脚本 | 说明 | 定时规则 |
|------|------|----------|
| `酷我音乐2099到期-修复版.py` | 签到、听歌、看广告、抽奖 | `0 8 * * *` |

### 环境变量
- `kwyy`: 账号信息
  - 单账号: `手机号#密码`
  - 备注格式: `备注#手机号#密码`
  - 多账号: `手机号1#密码1&手机号2#密码2`
- `KUWO_FREEMIUM_LOOP`: 免费听歌循环次数（默认1，最大10）

### 功能列表
1. 签到听歌任务链
2. 累计奖励任务
3. 开宝箱（普通+活动+时段补领）
4. 看视频拆红包（30次）
5. 整点领金币
6. 免费抽奖 + 广告抽奖
7. 观看惊喜广告
8. 免费听歌时长任务

### 使用方法
1. 在青龙面板中添加脚本
2. 设置环境变量 `kwyy`
3. 建议每天 8:00 运行
